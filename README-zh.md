# LM Downloader App Story

#### 介绍
主要目标是管理整个应用可以动态发布和加载的内容，包括应用介绍的程序，及相关数据库文件等。

#### 更新方案(每次启动时更新)
- 客户端App不自带数据库、前端和后端程序。
- 第一次启动时，显示加载中页面，在后台获取文件。
- 使用`git clone`获取`lm-downloader-app-story`，再执行`git pull`。
- 以上完成后启动本地服务，打开首页。


#### 目录
- db
- frontend
- server
- story-assets
